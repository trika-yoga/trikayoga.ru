Махавакья

महावाक्य
Mahāvākya
Великие изречения упанишад

प्रज्ञानम् ब्रह्म

prajñānam brahma

Чистое сознание (prajñānam) есть Абсолютная реальность (brahma). — «Айтарея-упанишада» 3.3 «Ригведы»

अहं ब्रह्मास्मि

ahaṃ brahmāsmi

Я (ahaṃ) есть (аsmi) Абсолютная реальность (brahma). — «Брихадараньяка-упанишада» 1.4.10 «Яджурведы»

तत् त्वम् असि / तत्त्वमसि

tat tvam asi

То (tat) ты (tvam) еси (asi). («Чхандогья-упанишада» 6.8.7 «Самаведы»)

अयम् आत्मा ब्रह्म

ayam ātmā brahma

Эта (ayam) душа (ātmā) есть Абсолютная реальность (brahma). («Мандукья-упанишад» 1.2 «Атхарваведы»)
